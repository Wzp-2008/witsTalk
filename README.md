## [WitsTalk]智慧语音

**本项目名称为:**`WitsTalk`
- `WitsTalk`是我们项目团队在2021/12/11制作的开源项目,项目的构思是在群语音的时候发现了一些缺点,才开发此项目.
- 实现在语音聊天中群员可以调节任何群员的`输出音量`以及自己的`输入音量`
- 管理员拥有最高权限,可以开关群员的`麦克风`,也可以调节群员的`输入音量` ...暂定是这些功能
- 项目的初衷是给minecraft玩家一个更舒服的语音环境,更好的交流环境
